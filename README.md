# 学生成绩管理系统

基于 node + express 开发的后台系统

# 注意
```
请确保你已安装以下应用:
1.node
2.mongodb
```

# 我的开发版本

	`node -v` -> v8.10.0
	`mongod -v` -> db version v3.6.3

## 拉取仓库
```
git clone git@github.com:hitore/stdsysapi.git
cd stdsysapi
```

## 安装依赖
```
npm install
or
cnpm install
or
yarn install
```

## 运行
```
node app
```
